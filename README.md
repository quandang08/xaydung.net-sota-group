# AMU Construction – High-End Industrial Website

**AMU Construction** is a premium website interface designed for a large-scale construction corporation, built around a **Solid & Industrial** design philosophy. The system employs a deep black base, combined with copper gold and brand red accents, to convey structural strength, reliability, and professional authority.

**Live Demo:** [amu-construction.vercel.app](https://amu-construction.vercel.app/)

---

## Key Features

* **Industrial UI**
  A minimalist, high-contrast interface inspired by technical blueprints and industrial design systems.

* **Cinematic Motion**
  Smooth, cinematic scrolling powered by **Lenis Scroll** and **GSAP**, delivering a refined visual flow across sections.

* **SVG Process Animation**
  Custom-drawn SVG diagrams illustrating construction workflows, animated progressively based on user scroll behavior.

* **Interactive Reveal Effects**
  News sections and imagery leverage hover-based interactions to enhance engagement and visual depth.

---

## Tech Stack

* **Styling**: Tailwind CSS
* **Animation**: GSAP (ScrollTrigger)
* **Smooth Scrolling**: Lenis
* **Icons**: Lucide Icons

## Setup Guide

### 1. Clone & Install Dependencies

```bash
git clone https://github.com/quandang08/xaydung.net-sota-group.git
cd xaydung.net-sota-group
npm install
```

### 2. Build Tailwind CSS

```bash
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```

### 3. Run the Project

Open `index.html` using Live Server or a local development server.
